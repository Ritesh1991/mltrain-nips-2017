This submission contains the code for this ICLR 2018 submission:
https://openreview.net/forum?id=Hksj2WWAW&noteId=Hksj2WWAW

This folder contains the poster: neuralMath.pdf
and a folder notebooks that contains two ipython notebooks for the poster.

File notebooks/nnTreeNotebook.ipynb performs training for the tables in the
poster and file notebooks/equationGenerator.ipynb contains the code for the
dataset generation scheme proposed in the paper.
